
## 简介
这是一个基于微信小程序的驾校约车系统，已经部署到云服务器上了。
## 基础设计方案
1. 数据库采用mysql数据库  
2. 系统分为学员端，教练端和管理员端。学员端和教练端在小程序上实现，管理员端在web端实现，提高管理员的录入效率。
3. 后台采用springboot+mybatis框架，前台采用微信小程序框架，bootstrap框架。

### 主要功能
* 学员教练信息的录入
* 课程分配
* 预约实现	
* 预约订单浏览			
*  教练评价             
* 驾驶技巧知识
* 驾校公告     
* 用户信息修改
